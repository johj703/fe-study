# fe-study

Git Convention

| 커밋 유형 | 이모지 | 설명                               |
| --------- | ------ | ---------------------------------- |
| feat      | ✨     | 새로운 기능 추가                   |
| init      | 🎉     | 프로젝트 초기 설정                 |
| fix       | 🐛     | 버그 수정                          |
| hotfix    | 🚑     | 긴급 버그 수정                     |
| style     | 💄     | CSS 등 사용자 UI 디자인 변경       |
| refactor  | ♻️     | 코드 리팩토링                      |
| docs      | 📝     | 문서 수정 (README.md 등)           |
| test      | 🍻     | 테스트 코드 추가 및 수정           |
| chore     | 🔨     | 빌드 설정 수정, npm 패키지 설치 등 |
| rename    | 🚚     | 파일/폴더 이름 변경 또는 이동      |
| remove    | 🔥     | 파일/기능 삭제                     |
| comment   | 💡     | 주석 추가 및 수정                  |
| git       | 🙈     | .gitignore 수정                    |
| !BREAKING | 💥     | 커다란 API 변경                    |
